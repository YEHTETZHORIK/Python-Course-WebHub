Data Types in Python

Python တွင် Data Types သည် အချက်အလက်များ၏ အမျိုးအစားကို သတ်မှတ်သည်။ Python တွင် အမျိုးအစားအလိုက် data types များစွာရှိပါသည်။ Data type များ၏ အခြေခံ အမျိုးအစားများမှာ အောက်ပါအတိုင်းဖြစ်ပါသည်:

Python Data Types

# Example of different data types
integer_value = 10          # Integer
float_value = 20.5         # Float
string_value = "Hello"     # String
boolean_value = True       # Boolean

print(integer_value)
print(float_value)
print(string_value)
print(boolean_value)

Result

10

20.5

Hello

True

ရှင်းပြချက်

Python တွင် အမျိုးအစားများအလိုက် variable တစ်ခုကို သတ်မှတ်နိုင်သည်။ အထက်ပါ example တွင် - integer_value သည် integer data type ဖြစ်ပြီး, float_value သည် float data type ဖြစ်ပါသည်။ string_value သည် string data type ဖြစ်ပြီး, boolean_value သည် boolean data type ဖြစ်ပါသည်။ ဒါတွေကို print() function ဖြင့် output အဖြစ် ကြည့်ရှုနိုင်ပါတယ်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. Data types အမျိုးမျိုးကို ပြန်လည်သတ်မှတ်ပြီး print() function ဖြင့် output ပြပါ။
  2. အထက်ပါ example တွင် ကိုယ်တိုင် values အသစ်များ ထည့်ပြီး result ကို အတူပြပါ။
