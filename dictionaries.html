Dictionaries in Python

Python တွင် Dictionary များသည် key-value pair များအဖြစ် အချက်အလက်များကို သိမ်းဆည်းထားရန် အသုံးပြုသော data structure ဖြစ်သည်။ Dictionary တွင် key များနှင့် value များကို pair အဖြစ်သတ်မှတ်ပြီး, dictionary ၏ key တွင် value တစ်ခုကို assign လုပ်နိုင်ပါသည်။

Dictionary Example

# Defining a dictionary in Python
person = {
    "name": "Zhorik",   # Key-value pair for name
    "age": 20,          # Key-value pair for age
    "city": "Yangon"    # Key-value pair for city
}

# Accessing values by keys
print(person["name"])  # Accessing value by key "name"
print(person["age"])   # Accessing value by key "age"
print(person["city"])  # Accessing value by key "city"

Result

Zhorik

20

Yangon

Dictionary Explanation

Python တွင် dictionary များကို curly braces ({}) အတွင်း key-value pair များကို define လုပ်သည်။ key သည် unique ဖြစ်ရမည်ဖြစ်ပြီး၊ value ကို key နှင့် အတူ assign လုပ်ရပါသည်။ ထို key-value pair များကို access လုပ်ရန် key ကို အသုံးပြုပါ။ အထက်ပါ example တွင် person dictionary တစ်ခု define လုပ်ပြီး, person["name"], person["age"], person["city"] ကို access လုပ်ထားပါသည်။

  2. dictionary ထဲမှ key-value pair များကို access လုပ်ပါ။
  3. value တစ်ခုကို update လုပ်ပါ။
