Functions in Python

Python တွင် Functions သည် code များကို reusable ဖြစ်စေပြီး, program တစ်ခုတည်းထဲတွင် လုပ်ဆောင်ချက်များကို ခွဲခြားစီမံနိုင်သည့် အစိတ်အပိုင်းများဖြစ်သည်။ Functions များသည် code ကို efficient နှင့် readable ဖြစ်စေပြီး, program ကို debugging လုပ်ရန်လည်း ပိုမိုလွယ်ကူစေပါသည်။

Function Example

# Defining a function in Python
def greet(name):
    print("Hello, " + name + "!")

# Calling the function
greet("Zhorik")

Result

Hello, Zhorik!

Function Explanation

Python တွင် function တစ်ခုကို def keyword ဖြင့်သတ်မှတ်ပြီး function name နဲ့ parameters များကို argument များအားလုံး ထည့်သွင်းထားနိုင်ပါတယ်။ function ကို call လုပ်ရန်လည်း function name နဲ့ parameters ကို ပြန်လည်ပေးပို့ရပါမည်။ အထက်ပါ example တွင် greet() function ကို "Zhorik" ဆိုသော string ကို parameter အဖြစ် ဖြည့်ထားပြီး, function ၏ output အဖြစ် "Hello, Zhorik!" ဟု print ထုတ်ပေးသည်။
