If Statements in Python

Python တွင် If statements များကို condition များအပေါ်အခြေခံပြီး program ရဲ့ control flow ကို လုပ်ဆောင်ရန် အသုံးပြုသည်။ If statement က တစ်ခုခု condition ဖြင့် true ဖြစ်ပါက သတ်မှတ်ထားသော code block ကို လုပ်ဆောင်ပေးပါသည်။

Example of If Statement

# If statement example
age = 20

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

Result

You are an adult

ရှင်းပြချက်

Python တွင် if statement ကို condition များကို စစ်ဆေးရာတွင် အသုံးပြုပါသည်။ အထက်ပါ example တွင် age ၏တန်ဖိုးသည် 18 နှင့်တူညီသည့်အတွက် if condition မှာ true ဖြစ်လာပြီး "You are an adult" ဆိုသော message ကို print() function ဖြင့် output ထုတ်ပြပါသည်။ Else statement ကိုလည်း အခြေအနေမမှန်လျှင် တရားဝင်သုံးနိုင်ပါသည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. age variable ကိုအသုံးပြုပြီး if statement ကို စမ်းပါ။
  2. number တန်ဖိုးများဖြင့် if-else statement များကို အသုံးပြုပြီး output တွေကို ကြည့်ပါ။
  3. condition များတွင် multiple conditions နှင့် logical operators (and, or) ကိုသုံးပါ။
