Lists in Python

Python တွင် List များသည် အချက်အလက်များကို အစီအစဉ်တကျ သိမ်းဆည်းထားရန် အသုံးပြုသော အရာများဖြစ်ပြီး, မည်သည့် data type ကိုမဆို သွင်းထားနိုင်သည်။ List တွင် string, integer, float, boolean အပါအဝင် အချက်အလက်မျိုးစုံ သိမ်းဆည်းနိုင်ပြီး၊ data များကို index လုပ်၍ အသုံးပြုနိုင်ပါသည်။

List Example

# Defining a list in Python
fruits = ["Apple", "Banana", "Cherry"]  # List containing strings

# Accessing elements from the list
print(fruits[0])  # First item
print(fruits[1])  # Second item
print(fruits[2])  # Third item

Result

Apple

Banana

Cherry

List Explanation

Python တွင် list များကို square brackets ([]) အတွင်းတွင် items များကို comma (,) ဖြင့် ခွဲခြား၍ သတ်မှတ်သည်။ List ၏ index သည် 0 အဖြစ်စတင်ပြီး, သင်သည် list ထဲရှိ item များကို index ဖြင့် access ပြုနိုင်သည်။ အထက်ပါ example တွင် fruits ဆိုသော list ကို define လုပ်ပြီး၊ fruits[0] သည် "Apple" ကို access လုပ်ပြီး fruits[1] နှင့် fruits[2] သည် "Banana" နှင့် "Cherry" တို့ကို access လုပ်သည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. သင်၏ကြိုက်နှစ်သက်သော fruits များကို list တွင်သိမ်းဆည်းပါ။
  2. list ထဲမှ item များကို index ဖြင့် access လုပ်ပြီး print() function ဖြင့် output ပြပါ။
  3. list ထဲမှ item တစ်ခုကို modify ပြီး list ၏ item အသစ်ကို ပြသပါ။
