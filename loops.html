For and While Loops in Python

Python တွင် loops သည် repetitive tasks များကို တိတိကျကျ လုပ်ဆောင်ရန် အသုံးပြုသည်။ For loop နှင့် While loop တို့သည် looping process တွေကို အလိုအလျောက် ပြုလုပ်ပေးနိုင်ပါသည်။

For Loop Example

# For loop example
for i in range(5):
    print(i)

Result (For Loop)

0

1

2

3

4

While Loop Example

# While loop example
count = 0
while count < 5:
    print(count)
    count += 1

Result (While Loop)

0

1

2

3

4

ရှင်းပြချက်

For loop သည် range() function ကို အသုံးပြု၍ အကွာအဝေးထဲမှ အချက်အလက်များကို loop ဖြင့် iterate လုပ်ပေးသည်။ ဥပမာ၊ range(5) သည် 0 မှ 4 အထိ values များကို generate လုပ်ပေးပါသည်။ အချို့သော situations တွင် While loop ကိုလည်း အသုံးပြုနိုင်ပြီး condition များအရ loop ကို run လေ့လာနိုင်ပါတယ်။ ထိုနောက် condition လိုအပ်သည့်အခါ loop ကို break ပြုလုပ်လေ့ရှိပါသည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. For loop ကို အသုံးပြုပြီး 1 မှ 10 အထိ numbers များကို print လုပ်ပါ။
  2. While loop ကို အသုံးပြုပြီး 0 မှ 10 အထိ odd numbers များကို print လုပ်ပါ။
  3. Loop ထဲမှာ break နဲ့ continue ကို အသုံးပြုပြီး loop behavior ကို control လုပ်ပါ။
