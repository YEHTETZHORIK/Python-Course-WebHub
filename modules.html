Modules and Libraries in Python

Python တွင် modules နှင့် libraries များသည် သင်၏ program အတွက် ပိုမိုအဆင့်မြှင့်သော functions နှင့် features များကို ထည့်သွင်းရန် အသုံးပြုသည်။ Module တစ်ခုသည် functions, variables, and classes များကို အစုလိုက်အသုံးပြုနိုင်သော file တစ်ခုဖြစ်သည်။ Libraries သည် modules များကို အုပ်စုထားပြီး သင်၏ application တွင် အရမ်းအသုံးဝင်သော tools များကို ပံ့ပိုးပေးသည်။

Module Example

# Importing the math module in Python
import math

# Using math module to calculate square root
number = 16
result = math.sqrt(number)
print(result)

Result

4.0

Explanation

Python တွင် module များကို import လုပ်ပြီး သင်၏ program အတွင်းရှိ functions များကို အသုံးပြုနိုင်သည်။ အထက်ပါ example တွင် math module ကို import လုပ်ပြီး, sqrt() function ကို အသုံးပြုပြီး 16 ရဲ့ square root ကို တွက်ချက်ထားသည်။ အောက်ပါ import math သည် math module ကို program ၏အတွင်းသို့ load လုပ်ပေးပြီး၊ math.sqrt(number) သည် 16 ၏ square root ကို return လုပ်သည်။

Using Other Libraries

Python တွင် နာမည်ကြီး libraries များဖြစ်သော numpy, pandas, requests များကို ရယူပြီး သင်၏ project တွင် အသုံးပြုနိုင်ပါသည်။ ပရော်ဖက်ရှင်နယ် data analysis, web scraping, machine learning စသည့် tasks များကို ပြုလုပ်ရာတွင် libraries များကို အထူးသဖြင့် အသုံးပြုသည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. math module ကို import လုပ်ပြီး square root, exponential value, or pi value များကို ထုတ်ယူပါ။
  2. random module ကို import လုပ်ပြီး random number တစ်ခု generate လုပ်ပါ။
  3. os module ကို import လုပ်ပြီး system-related functions များကို အသုံးပြုပါ။
