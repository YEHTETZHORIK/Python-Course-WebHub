Operators in Python

Python တွင် Operators များသည် အချက်အလက်များကို လုပ်ဆောင်ရန် အသုံးပြုသော စက်ရုပ်များဖြစ်သည်။ Operators များကို Arithmetic, Comparison, Logical, Assignment နှင့် Bitwise အစရှိသော အမျိုးအစားများဖြင့် စီမံခန့်ခွဲနိုင်ပါသည်။

Example of Operators in Python

# Arithmetic Operators
x = 10
y = 5

# Addition
addition = x + y  # 10 + 5
# Subtraction
subtraction = x - y  # 10 - 5
# Multiplication
multiplication = x * y  # 10 * 5
# Division
division = x / y  # 10 / 5

print("Addition:", addition)
print("Subtraction:", subtraction)
print("Multiplication:", multiplication)
print("Division:", division)

Result

Addition: 15

Subtraction: 5

Multiplication: 50

Division: 2.0

ရှင်းပြချက်

Python တွင် Arithmetic Operators များကို သုံးပြီး သင့်ရဲ့ values တွေကို တွက်ချက်နိုင်ပါသည်။ အထက်ပါ example တွင် addition, subtraction, multiplication နှင့် division operators များကို ပြသထားပါသည်။ ထို operators များအားလုံးမှာ x နှင့် y ကို operands အဖြစ် အသုံးပြုထားပြီး, ပြသထားသော result တွေကို print() function ဖြင့် output ပြထားပါသည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. Arithmetic operators များဖြင့် addition, subtraction, multiplication နှင့် division operations ပြုလုပ်ပါ။
  2. Comparison operators (==, !=, <, >, <=, >=) ကိုသုံးပြီး comparisons ပြုလုပ်ပါ။
  3. Logical operators (and, or, not) ကိုအသုံးပြုပြီး conditionals တွေကို စမ်းပါ။
