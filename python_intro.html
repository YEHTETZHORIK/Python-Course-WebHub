What is Python?

Python သည် အလွန်လွယ်ကူပြီး အစွမ်းထက်သော programming language တစ်ခုဖြစ်ပါသည်။ Python ကို 1991 ခုနှစ်တွင် Guido van Rossum မှ ဖန်တီးခဲ့ပြီး လက်ရှိတွင် AI (Artificial Intelligence), Machine Learning, Data Science, Web Development စသောနယ်ပယ်များတွင် အသုံးပြုနေပါသည်။

Python Example

# This is a simple Python program
print("Hello, World!")

Result

Hello, World!

ရှင်းပြချက်

Python code ရေးသားရန် text editor များဖြင့် ပုံမှန် Python file (.py) တစ်ခုဖန်တီးပါ။ အထက်ပါ example မှာ "Hello, World!" ဆိုသောစာသားကို console မှာ ပြပါသည်။ Python ကို command line သို့မဟုတ် IDE (Integrated Development Environment) အသုံးပြု၍ run လိုက်လျှင် အဆိုပါ output ကို ကြည့်ရှုနိုင်ပါသည်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. Python code တွင် print() function ကို အသုံးပြု၍ မိမိကိုယ်ကို မိတ်ဆက်ပါ။
  2. သင်၏ code ကို .py ဖိုင်အဖြစ် သိမ်းပြီး terminal သို့မဟုတ် IDE တွင် run ပြုလုပ်ပါ။
