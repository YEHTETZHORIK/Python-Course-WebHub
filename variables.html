Variables in Python

Python တွင် Variables သည် အချက်အလက်များကို သိမ်းဆည်းထားရန် အသုံးပြုသည့် အကြောင်းအရာများဖြစ်သည်။ Variable ကို value တစ်ခု assign (သတ်မှတ်) ပြီး သင့်ရဲ့ program မှာ ထပ်မံအသုံးပြုနိုင်ပါတယ်။

Variable Example

# Defining variables in Python
name = "Zhorik"  # String variable
age = 20         # Integer variable

print(name)
print(age)

Result

Zhorik

20

ရှင်းပြချက်

Python တွင် variable များကို assignment လုပ်ရာတွင် variable name (အသုံးပြုလိုသော အမည်) နဲ့ value (သတ်မှတ်လိုသော တန်ဖိုး) ကို အသုံးပြုပါသည်။ အထက်ပါ example တွင် name ဆိုတဲ့ variable တွင် "Zhorik" ဆိုသော string value ကို assign လုပ်ခဲ့ပြီး, age variable တွင် 20 ဆိုသော integer value ကို assign လုပ်ခဲ့ပါသည်။ အဲဒီ variable များကို print() function ဖြင့် output အဖြစ် ကြည့်ရှုနိုင်ပါတယ်။

လေ့ကျင့်ခန်းများ

အောက်ပါ လေ့ကျင့်ခန်းများကို ပြုလုပ်ပါ -

  1. name နှင့် age variable များကို သတ်မှတ်ပြီး print() function ဖြင့် မိမိအကြောင်းအရာများကို ပြပါ။
  2. ပြုလုပ်ထားသော variables များကို console တွင် output အဖြစ် ကြည့်ပါ။
